еҶІе•ҠпјҒ

жңҖиҝ‘з»ҸеёёзңӢгҖҠеӣӣй©ұе…„ејҹгҖӢпјҢи„‘еӯҗе·Із»Ҹиў«вҖңеҶІе•ҠвҖқжҙ—жҺүдәҶгҖӮ

вҖңеҶІе•ҠпјҢе°ұи®©жҲ‘д»¬дёҖи·ҜйўҶе…ҲеҲ°еә•еҗ§пјҒвҖқпјҢвҖңеҶІе•ҠпјҢзңӢжҲ‘зҡ„иғңеҲ©еҶІй”ӢпјҒвҖқпјҢвҖңеҶІе•ҠпјҒе…Ҳй©ұйҹійҖҹпјҒвҖқпјҢвҖңеҺ»еҗ§пјҒдёүи§’з®ӯпјҒвҖқпјҢвҖңеҶІе•ҠпјҒз–ҫйҖҹж–§еӨҙпјҒвҖқпјҢвҖңеҶІе•ҠпјҒдәҢйғҺдёёзү№еҲ«еҸ·пјҒвҖқпјҢвҖңеҶІе•ҠпјҒвҖқ

з»ҲдәҺжқҘеҲ°дәҶи®ҫи®ЎжЁЎејҸзҡ„вҖңдәәжҖ§вҖқйғЁеҲҶпјҢиЎҢдёәеһӢжЁЎејҸпјҢдёәд»Җд№ҲиҜҙиЎҢдёәеһӢжЁЎејҸжҳҜвҖңдәәжҖ§вҖқйғЁеҲҶе‘ўпјҢеӣ дёәиЎҢдёәеһӢжЁЎејҸеҪ“дёӯзҡ„ 11 з§Қи®ҫи®ЎжЁЎејҸеҜ№зҗҶи§ЈйғҪйқһеёёзҡ„еҸӢеҘҪе•ҠгҖӮжүҖд»ҘжҺҘдёӢжқҘзҡ„еҶ…е®№еҸҜиғҪи®©жҲ‘еӯҰиө·жқҘиҜҙдёҚе®ҡжӣҙжңүи¶ЈдәӣгҖӮ

жЁЎжқҝж–№жі•жЁЎејҸ

е®ҡд№үдёҖдёӘж“ҚдҪңдёӯзҡ„з®—жі•йӘЁжһ¶пјҢе°Ҷз®—жі•зҡ„дёҖдәӣжӯҘйӘӨе»¶иҝҹеҲ°еӯҗзұ»дёӯпјҢдҪҝеҫ—еӯҗзұ»еңЁеҸҜд»ҘдёҚж”№еҸҳиҜҘз®—жі•з»“жһ„зҡ„жғ…еҶөдёӢйҮҚе®ҡд№үиҜҘз®—жі•зҡ„жҹҗдәӣзү№е®ҡжӯҘйӘӨгҖӮ

еҰӮдҪ•зҗҶи§Ј

е®ҡд№үдёҖдёӘж“ҚдҪңдёӯзҡ„з®—жі•йӘЁжһ¶пјҢиҜҙзҷҪдәҶиҝҷдёҚе°ұжҳҜдёҖдёӘжӯҘйӘӨзәҰжқҹеҗ—пјҹеңЁзңӢз¬¬дәҢж®өпјҢе°Ҷз®—жі•зҡ„дёҖдәӣжӯҘйӘӨе»¶иҝҹеҲ°еӯҗзұ»дёӯпјҢж„ҸжҖқе°ұжҳҜжӯҘйӘӨйҮҢзҡ„дёҖйғЁеҲҶз•ҷз»ҷдҪ дәҶпјҢе…·дҪ“жҖҺд№ҲеҒҡзңӢдҪ пјҲеӯҗзұ»пјүиҮӘе·ұдәҶгҖӮдҪҝеҫ—еӯҗзұ»еңЁеҸҜд»ҘдёҚж”№еҸҳиҜҘз®—жі•з»“жһ„зҡ„жғ…еҶөдёӢйҮҚе®ҡд№үиҜҘз®—жі•зҡ„жҹҗдәӣзү№е®ҡжӯҘйӘӨпјҢиҝҷеҸҘе°ұжӣҙеҘҪзҗҶи§ЈдәҶпјҢжңүдәҶжӯҘйӘӨзҡ„зәҰжқҹпјҢдҪ иҙҹиҙЈжү§иЎҢе…·дҪ“зҡ„жӯҘйӘӨпјҢиҜҙзҷҪдәҶпјҢжӯҘйӘӨеҸӘиҰҒжү§иЎҢе°ұеҸҜд»ҘпјҢдёҚз®ЎдҪ жҖҺд№ҲеҒҡпјҢжүҖд»Ҙд№ҹе°ұжңүдәҶдёҚж”№еҸҳз»“жһ„зҡ„жғ…еҶөдёӢеҸҜд»ҘйҮҚж–°е®ҡд№үзү№е®ҡзҡ„жӯҘйӘӨпјҢиҝҷйҮҢзҡ„зү№е®ҡжҢҮзҡ„е°ұжҳҜзәҰжқҹжӯҘйӘӨйҮҢз•ҷз»ҷдҪ зҡ„йӮЈйғЁеҲҶжӯҘйӘӨгҖӮ

  • жҜ”еҰӮжҲ‘д»¬з»ҸеёёеҶҷ PPT зҡ„е°ҸдјҷдјҙзҹҘйҒ“пјҢеңЁ PPT дёӯпјҢзӮ№еҮ»ж–°еўһдёҖйЎө PPT пјҢеҲӣе»әеҮәжқҘзҡ„йЎөйқўпјҢеӨ§йғЁеҲҶзҡ„еёғеұҖж јејҸйғҪжҳҜе·Із»Ҹи®ҫзҪ®еҘҪзҡ„пјҢжҲ‘д»¬еҸӘйңҖиҰҒеҺ»еЎ«е……йҮҢйқўзҡ„еҶ…е®№е°ұеҸҜд»ҘдәҶгҖӮ
  • еҶҚжҜ”еҰӮз»ҸеёёеҶҷйңҖжұӮиҜҙжҳҺд№Ұзҡ„ж—¶еҖҷпјҢдјҡе’ҢеҗҢдәӢиҰҒдёҖд»ҪвҖңжЁЎжқҝвҖқпјҢиҝҷд№ҹжҳҜдёҖз§ҚжЁЎжқҝж–№жі•жЁЎејҸзҡ„дҪ“зҺ°гҖӮ
  • иҝҳжңүеҰӮжһңдҪ дҪҝз”ЁиҝҮ maven еҲӣе»әиҝҮйЎ№зӣ®пјҢйӮЈиҝҷдёӘдҪ дёҖе®ҡдёҚйҷҢз”ҹ

йҮҚж–°е®ҡд№ү

жЁЎжқҝж–№жі•жЁЎејҸе°ұжҳҜпјҡжҸҗдҫӣдёҖдёӘе…·дҪ“зҡ„жӯҘйӘӨпјҢ1,2,3,4пјҢзҺ°еңЁ1,2,4йғҪеҶҷеҘҪдәҶпјҢжӯҘйӘӨ3зҡ„е®һзҺ°з”ұе…·дҪ“зҡ„жү§иЎҢиҖ…иҜҙдәҶз®—пјҢеҸӘиҰҒжӯҘйӘӨз¬ҰеҗҲиҰҒжұӮпјҢйҡҸдҪ еҸ‘жҢҘгҖӮ

жЁЎжқҝж–№жі•жЁЎејҸзұ»еӣҫ рҹ“Ң

зңӢзұ»еӣҫе°ұжҳҜжҠҪиұЎе’Ңеӯҗзұ»зҡ„ж ·еӯҗпјҢиҖҢе®һйҷ…жЁЎжқҝж–№жі•жЁЎејҸе°ұжҳҜеҲ©з”ЁдәҶ"жҠҪиұЎ"гҖӮжҳҜдёҖдёӘе®Ңе…ЁйҒөе®ҲејҖй—ӯеҺҹеҲҷзҡ„и®ҫи®ЎжЁЎејҸгҖӮеҸҜд»Ҙиҝҷд№ҲиҜҙпјҢжҗһжҮӮдәҶжЁЎжқҝж–№жі•жЁЎејҸпјҢејҖй—ӯеҺҹеҲҷеҹәжң¬е°ұйҖҡдәҶгҖӮ

жіЁж„ҸпјҡжЁЎжқҝж–№жі•жЁЎејҸдёӯдёҺжҲ‘д»¬е№іж—¶з»§жүҝжҠҪиұЎзұ»жңүдёҖдёӘе…ій”®жҖ§зҡ„еҢәеҲ«пјҢе°ұжҳҜе…ҘеҸЈж–№жі•пјҢжӯЈеёёжҠҪиұЎзұ»з»§жүҝжҳҜдёҚйңҖиҰҒжңүиҝҷдёӘжүҖи°“зҡ„е…ҘеҸЈж–№жі•пјҢеҸҜд»ҘйҖҡиҝҮе…ҘеҸЈж–№жі•жқҘзЎ®е®ҡз®—жі•зҡ„жү§иЎҢйЎәеәҸпјҢеҚіз®—жі•йӘЁжһ¶гҖӮ

жЁЎжқҝж–№жі•жЁЎејҸеҸҜд»ҘиҜҙжҳҜйқһеёёз®ҖеҚ•зҡ„дёҖз§Қи®ҫи®ЎжЁЎејҸдәҶпјҢиҷҪз„¶з®ҖеҚ•пјҢдҪҶе®ғзҡ„дҪңз”ЁеҚҙеҫҲеӨ§гҖӮжҜ”еҰӮжҲ‘д»¬з»ҸеёёдҪҝз”Ёзҡ„lock й”ҒпјҢе®ғзҡ„е®һзҺ°е°ұеҲ©з”ЁдәҶ AQS пјҢиҖҢ AQS е°ұжҳҜдҪҝз”Ё жЁЎжқҝж–№жі•жЁЎејҸ з»ҙжҠӨзҡ„дёҖдёӘй”ҒжЎҶжһ¶пјҢйҖҡиҝҮе®ғеҸҜд»Ҙеҝ«йҖҹзҡ„ејҖеҸ‘еҮәдёҖдёӘй”ҒгҖӮиҝҷжӯҘеҸҜд»Ҙз»“еҗҲ AQS зҡ„д»Јз ҒжқҘзңӢдёҖзңӢгҖӮ

        int threadCount = Runtime.getRuntime().availableProcessors();
        CountDownLatch signal = new CountDownLatch(threadCount);
        int loop = 100000;
        for (int i = 0; i < threadCount; i++) {
            Thread thread = new Thread(() -> {
                int l = 0;
                while (l < loop) {
                    lock.lock();
                    try {
                        f++;
                    } catch (Exception e) {

                    } finally {
                        lock.unlock();
                    }
                    l++;
                }
                signal.countDown();
            });
            thread.start();
        }
        signal.await();
        Assertions.assertEquals(threadCount * loop, f);
    }

иҝҗиЎҢз»“жһң

иҝҷйҮҢе»әи®®еӨ§е®¶й…ҚеҗҲжәҗз ҒеӯҰд№ пјҢеҗҢж—¶д№ҹиғҪеӯҰд№ дёҖдәӣдёҺй”Ғе®һзҺ°дёҠзҡ„дёҖдәӣзӣёе…ізҹҘиҜҶпјҢеҰӮжһңжңүдёҚжё…жҘҡжҲ–и§үеҫ—жңүз–‘й—®зҡ„ең°ж–№пјҢж¬ўиҝҺеҠ жҲ‘еҫ®дҝЎдёҖиө·и®Ёи®әпјҲlvgoccпјү

жҖ»з»“ рҹ“ҡ

йҖҡиҝҮжЁЎзүҲжЁЎејҸеҸҜд»Ҙе°ҶдёҖдәӣжҲ‘д»¬жғіиҰҒзәҰжқҹзҡ„жү§иЎҢжӯҘйӘӨеӣәе®ҡдёӢжқҘпјҢеҗҢж—¶еҜ№дәҺжӯҘйӘӨдёӯйҮҚеӨҚзҡ„еҶ…е®№еҸҜд»ҘиҝӣиЎҢжҠҪиұЎпјҢиҝҷж ·е°ұеҸҜд»Ҙз®ҖеҢ–еҫҲеӨҡеӯҗзұ»зҡ„ж“ҚдҪңпјҢд№ҹйҒҝе…ҚдәҶдёҖдәӣдёҚеҝ…иҰҒзҡ„еҶ—дҪҷд»Јз Ғдә§з”ҹгҖӮ

дҪҝз”ЁжЁЎжқҝж–№жі•жЁЎејҸеҸҜд»Ҙйқһеёёз®ҖеҚ•зҡ„жқҘзәҰжқҹдёҖж®өйҖ»иҫ‘зҡ„жү§иЎҢиҰҒжұӮгҖӮеңЁеҒҡзЁӢеәҸжү©еұ•йҷҗеҲ¶ж—¶пјҢйқһеёёжңүз”ЁгҖӮе®ҡд№үеҘҪе…·дҪ“зҡ„йҖ»иҫ‘жөҒзЁӢжҠҪиұЎзұ»пјҢе°Ҷе…¬е…ұйғЁеҲҶд»Јз ҒеҶҷеңЁжҠҪиұЎзұ»дёӯпјҢз„¶еҗҺе°Ҷе…¶дёӯйңҖиҰҒдҪҝз”ЁиҖ…иҮӘиЎҢе®һзҺ°зҡ„ж–№жі•е®ҡд№үдёәжҠҪиұЎж–№жі•пјҢиҝҷж ·еҪ“д»–з»§жүҝиҝҷдёӘзұ»зҡ„ж—¶еҖҷпјҢеҸӘйңҖиҰҒе°ҶеҜ№еә”зҡ„жҠҪиұЎж–№жі•е®һзҺ°е°ұеҘҪдәҶпјҢдёҚйңҖиҰҒе…ізі»е…¶е…·дҪ“зҡ„жү§иЎҢйЎәеәҸгҖӮ

дҪҶеҗҢж ·зҡ„пјҢиҝҷж ·дјҡдҪҝжү§иЎҢйЎәеәҸеҜ№дҪҝз”ЁиҖ…е®Ңе…ЁйҖҸжҳҺпјҢеҰӮжһңжҠҪиұЎзҡ„ж–№жі•иҫғдёәеӨҚжқӮж—¶пјҢеҜ№дәҺдёҖдёӘеҲқж¬ЎдҪҝз”ЁиҜҘйҖ»иҫ‘зҡ„дәәжқҘиҜҙпјҢеҮәзҺ° bug еҸҜиғҪдјҡдҪҝд»–еҫҲвҖңйҡҫеҸ—вҖқпјҢеӣ дёәд»–йңҖиҰҒжҗһжё…жҘҡж•ҙдёӘжҠҪиұЎзұ»дёӯжҜҸдёӘж–№жі•зҡ„жү§иЎҢйЎәеәҸжүҚиғҪжӣҙеҘҪзҡ„еҺ»и§ЈеҶій—®йўҳгҖӮиҝҷдёҖзӮ№пјҢеўһеҠ дәҶзі»з»ҹзҡ„еӨҚжқӮжҖ§гҖӮдёҚиҝҮпјҢи®ҫи®ЎжЁЎејҸзҡ„еӨҚжқӮжҖ§пјҢжҳҜдёҚеҸҜйҒҝе…Қзҡ„гҖӮеңЁеҠҹиғҪеӨҚз”ЁгҖҒжҸҗй«ҳз”ҹдә§еҠӣдёҠжқҘиҜҙпјҢиҝҷзӮ№еӨҚжқӮжҖ§зҡ„д»Јд»·пјҢиҝҳжҳҜиғҪеӨҹжҺҘеҸ—зҡ„гҖӮ
